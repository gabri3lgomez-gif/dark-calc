# Dark Calc

A dark-themed calculator application built as a Trusted Web Activity (TWA).

## Features

- Dark theme UI
- Fast calculations
- Android APK available

## Build Status

![Build Android APK](https://github.com/gabri3lgomez-gif/dark-calc/workflows/Build%20Android%20APK%20(TWA)/badge.svg)

## Installation

Download the APK from the [Releases](https://github.com/gabri3lgomez-gif/dark-calc/releases) section.

## Development

This project uses GitHub Actions to automatically build Android APKs.

For more information, see the workflow configuration in `.github/workflows/build-android-apk.yml`.
